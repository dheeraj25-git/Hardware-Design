## Overview

A curated collection of **open-source PCB designs** for:
- **Wireless communication** (2.4 GHz, Sub-GHz, LoRa)
- **USB interfaces** (dongles, bridges, HID)
- **Microcontroller dev boards** (STM32, RP2040, ESP32)
- **Sensor modules** (IMU, environmental, RF)
- **Power & battery management**

All projects are designed in **KiCad**, fully documented, and ready for **prototyping or production**.
